# Hotkeys

This is a list of hotkeys for when the game is on **Hotkey Mode**. Use **Tab** to switch.

| Key | Command |
| ------- | -------------------------------- |
| Tab | Switch Macro and Hotkey Mode |
| Escape | Fullscreen |
| Space | Zoom/Scope / Use iron sights / Tank gun |
| W / North | Move Up |
| S / South | Move Down |
| A / West | Move Left |
| D / East | Move Right |
| Q, CTRL+Q | Drop item |
| E, CTRL+E | Quick equip |
| R, CTRL+R | Throw |
| T | Say (IC chat) |
| X, CTRL+X | Swap hands |
| Z, CTRL+Z, CTRL+Y | Activate held object |
| C, CTRL+C | Secondary activation (item/vehicle brake) |
| B | Buckle/Unbuckle |
| H, CTRL+H | Holster/Unholster |
| F, CTRL+F | Change Intent (cycle left) |
| G, CTRL+G | Change Intent (cycle right) |
| 1, CTRL+1 | Help Intent |
| 2, CTRL+2 | Disarm Intent |
| 3, CTRL+3 | Grab Intent |
| 4, CTRL+4 | Harm Intent |
| 5 | Quick-cast spell 5 |
| 6 | Quick-cast spell 6 |
| 7 | Quick-cast spell 7 |
| 8 | Quick-cast spell 8 |
| 9 | Quick-cast spell 9 |
| J, CTRL+J | Change Gun Mode (Aim/Fire) |
| L, CTRL+L | Attack in facing direction |
| M | Toggle run/walk |
| N | Change Tactic (Aim/Rush/Charge/Defense) |
| U | Change Secondary Attack (Bite/Jump/Kick) |
| V | Change Defense (Parry/Dodge) |
| Y | Activate held object |
| . | Cycle movement mode |
| , | Prone/Rest |
| Insert | Cycle intents (right) |
| Delete | Stop Pulling |
| O | OOC chat |
| ALT+L | LOOC chat |
| F1 | Adminhelp |
| F2 | Mentorhelp |
| F3 | Emote (/me) |
| F12 | Change HUD / Cursor style |
| Numpad 1 | Select Right Leg (targeting) |
| Numpad 2 | Select Groin (targeting) |
| Numpad 3 | Select Left Leg (targeting) |
| Numpad 4 | Select Right Arm (targeting) |
| Numpad 5 | Select Chest (targeting) |
| Numpad 6 | Select Left Arm (targeting) |
| Numpad 8 | Select Head (targeting) |
